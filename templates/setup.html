VeilX

Инструкция по подключению

Выбери свою платформу и следуй шагам.

iOS — FoXray

Скачать: App Store

  1. Установи FoXray из App Store по ссылке выше
  2. Открой свою персональную страницу VeilX (ссылку тебе отправил админ)
  3. Нажми кнопку «Скопировать конфиг» на своей странице
  4. Открой приложение FoXray
  5. Нажми «+» (внизу справа) → выбери «Добавить из буфера обмена»
  6. Появится новый сервер в списке — нажми на него, затем нажми кнопку ▶ (Play) для подключения
  7. Разреши VPN-подключение, если iOS попросит
  8. Готово! Проверь на ip.me — IP должен быть не российский

Альтернативный способ: вместо копирования можно открыть камеру в FoXray (иконка QR) и отсканировать QR-код со страницы VeilX.

Android — v2rayNG

Скачать: Google Play / GitHub APK

Если Google Play недоступен — скачай APK-файл с GitHub (файл с названием v2rayNG_x.x.x.apk).

  1. Установи v2rayNG из Google Play или скачай APK по ссылке выше
  2. Открой свою персональную страницу VeilX (ссылку тебе отправил админ)
  3. Нажми кнопку «Скопировать конфиг» на своей странице
  4. Открой приложение v2rayNG
  5. Нажми «+» (вверху справа) → выбери «Импорт конфигурации из буфера обмена»
  6. Появится сервер в списке — нажми большую круглую кнопку «V» внизу экрана для подключения
  7. Разреши VPN-подключение, если Android попросит
  8. Готово! Проверь на ip.me — IP должен быть не российский

Альтернативный способ: вместо копирования можно в v2rayNG нажать «+» → «Сканировать QR-код» и навести камеру на QR-код со страницы VeilX.

Windows — v2rayN

Скачать: v2rayN (GitHub)

Скачай файл v2rayN-windows-64-With-Core.zip из последнего релиза.

  1. Скачай архив v2rayN по ссылке выше и распакуй в любую папку
  2. Запусти v2rayN.exe из распакованной папки
  3. Открой свою персональную страницу VeilX в браузере (ссылку тебе отправил админ)
  4. Нажми кнопку «Скопировать конфиг» на своей странице
  5. В v2rayN: нажми «Серверы» в меню → «Импорт серверов из буфера обмена»
  6. Появится сервер в списке — кликни правой кнопкой по иконке v2rayN в трее (возле часов) и выбери «Системный прокси» → «Включить»
  7. Готово! Проверь на ip.me — IP должен быть не российский

Совет: если v2rayN не запускается — попробуй запустить от имени администратора (правая кнопка → «Запуск от имени администратора»).

macOS — FoXray

Скачать: FoXray (App Store)

FoXray работает и на iPhone, и на Mac. Это одно приложение.

  1. Установи FoXray из App Store (Mac) по ссылке выше
  2. Открой свою персональную страницу VeilX в браузере (ссылку тебе отправил админ)
  3. Нажми кнопку «Скопировать конфиг» на своей странице
  4. Открой приложение FoXray
  5. Нажми «+» → выбери «Добавить из буфера обмена»
  6. Появится сервер в списке — нажми на него, затем нажми ▶ (Play) для подключения
  7. Разреши VPN-подключение, если macOS попросит
  8. Готово! Проверь на ip.me — IP должен быть не российский

Не работает?

  • Конфиг не добавляется в приложение — убедись, что ты нажал «Скопировать конфиг» на своей странице VeilX и скопировал ссылку полностью (она начинается с vless://)
  • Подключение не устанавливается — попробуй другой клиент из списка выше. Например, на Android вместо v2rayNG можно использовать Nekobox
  • Подключился, но сайты не открываются — проверь на ip.me, изменился ли IP. Если нет — отключи и подключи VPN заново
  • Ничего не помогает — напиши админу в Telegram, он поможет разобраться
